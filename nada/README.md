# NADA - Data Catalog

NADA is an open source microdata cataloging system, compliant with the Data Documentation Initiative (DDI) and Dublin Coreâ€™s RDF metadata standards. It serves as a portal for researchers to browse, search, compare, apply for access, and download relevant census or survey datasets, questionnaires, reports and other information.

## Getting Started


### Server Requirements

* PHP version 7 or later
* MySQL or Microsoft SQL Server database
* Apache, IIS or NGINX

### Installation

Please see the [NADA installation guide](https://nada.readthedocs.io/en/v5.0/installation-guide.html).


### Upgrading from older versions of NADA

Please see the [NADA upgrade guide](https://nada.readthedocs.io/en/v5.0/upgrade-from-4.4-to-5.html).
